# 🚀 Portfólio Igor Elias - HTML/CSS/JS Puro

Portfólio profissional desenvolvido com HTML, CSS e JavaScript vanilla (sem frameworks).

## 📋 Sobre o Projeto

Este é um portfólio moderno e responsivo que apresenta minhas habilidades, projetos e informações de contato. Foi desenvolvido do zero usando apenas tecnologias web nativas para garantir máxima performance e compatibilidade.

## ✨ Funcionalidades

- ✅ Design responsivo (mobile-first)
- ✅ Internacionalização (i18n) - Suporte a PT-BR e EN
- ✅ Tema escuro (padrão) / claro com persistência e prevenção de "flash"
- ✅ Animações suaves e scroll reveals
- ✅ Efeito typewriter no hero
- ✅ Sistema de partículas no background
- ✅ Formulário de contato com validação e feedback
- ✅ Smooth scroll entre seções
- ✅ SEO otimizado com Open Graph tags
- ✅ Performance otimizada (Lazy Loading, Icon fonts)

## 🛠️ Tecnologias Utilizadas

- **HTML5** semântico
- **CSS3** (Custom Properties, Grid, Flexbox, Animations)
- **JavaScript ES6+** (Vanilla JS, Modules)
- **Internacionalização** (Sistema de i18n customizado)
- **Google Fonts** (Inter)
- **Iconografia**: Lucide Icons & Simple Icons

## 📁 Estrutura de Arquivos

```
portfolio/
├── index.html              # Página principal
├── assets/
│   ├── css/
│   │   ├── reset.css      # Reset CSS
│   │   ├── variables.css  # Variáveis CSS e Tokens de Design
│   │   ├── base.css       # Estilos globais e base do site
│   │   ├── components.css # Componentes reutilizáveis
│   │   ├── animations.css # Biblioteca de animações
│   │   └── main.css       # Estilos específicos de seções
│   ├── js/
│   │   ├── utils/
│   │   │   ├── theme.js   # Gerenciamento de tema
│   │   │   ├── i18n.js    # Sistema de tradução (PT/EN)
│   │   │   └── dom.js     # Utilitários DOM
│   │   ├── components/
│   │   │   ├── typewriter.js # Lógica do efeito de digitação
│   │   │   ├── scroll.js     # Observadores de scroll e animações de entrada
│   │   │   ├── form.js       # Validação e envio do formulário
│   │   │   └── particles.js  # Animação de fundo (Canvas)
│   │   └── main.js        # Script de inicialização
│   └── images/            # Assets de imagem (Portfolio e Projetos)
├── favicon.svg            # Ícone do site
└── README.md              # Documentação do projeto
```

## 🚀 Como Usar

1. Clone ou baixe o repositório
2. Abra o arquivo `index.html` em um navegador moderno
3. Ou use um servidor local:

   ```bash
   # Com Python
   python -m http.server 8000

   # Com Node.js (http-server)
   npx http-server

   # Com PHP
   php -S localhost:8000
   ```
